Here's a cool tool I found while poking around looking at how to display my GitHub streak!


Readme Stats: This project, Anuraghazra Repo uses the GitHub API to fetch and display contribution streaks in your README file. You can embed it in your website as well.

Example of the URL for a streak:

https://github-readme-streak-stats.herokuapp.com/?user=yourusername

This service automatically updates every day, and you can customize it with parameters for appearance.
